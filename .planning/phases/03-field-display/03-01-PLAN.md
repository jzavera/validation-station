# Plan 03-01: Field Display Components

**Phase:** 03-field-display
**Started:** 2026-02-18
**Status:** In Progress
**Requirements:** FIELD-01, FIELD-02, FIELD-03, FIELD-04, FIELD-05, FIELD-06

## Goal

Build all field display components so the operator can see extracted fields organized by groups with confidence indicators, confirmed/missing states, and active field highlighting.

## Tasks

<task id="1" status="pending">
Create `src/utils/confidence.ts` utility with `getConfidenceColor()` and `getConfidenceLabel()` helpers that return Tailwind color classes and human-readable labels based on confidence thresholds (red <0.7, yellow 0.7-0.9, green >=0.9).
</task>

<task id="2" status="pending">
Create `src/components/ConfidenceBadge.tsx` component that renders a colored dot based on confidence score, a green checkmark for confirmed fields, or a gray dash for missing fields.
</task>

<task id="3" status="pending">
Create `src/components/FieldRow.tsx` component that displays a single field row with ConfidenceBadge, field label, extracted value, click-to-select handler dispatching SET_ACTIVE_FIELD, and blue ring highlight when active.
</task>

<task id="4" status="pending">
Create `src/components/FieldGroupCard.tsx` component that renders a group header label and a list of FieldRow children for each field in the group.
</task>

<task id="5" status="pending">
Create `src/components/FieldPanel.tsx` component that maps through fieldGroups from state and renders a scrollable container of FieldGroupCard components.
</task>

<task id="6" status="pending">
Update `src/App.tsx` to replace the right panel placeholder with the FieldPanel component.
</task>

<task id="7" status="pending">
Verify `npx tsc --noEmit` and `npm run build` both pass with no errors. Visually confirm field groups render with correct confidence indicators in the browser.
</task>

## Acceptance Criteria

1. Fields appear in the right panel grouped under labeled section headers matching field group names
2. Each field row shows a confidence dot (red <0.7, yellow 0.7-0.9, green >=0.9), the field label, and the extracted value
3. Confirmed fields display a green checkmark instead of the confidence dot
4. Missing fields show a dash icon with gray styling
5. Clicking a field row highlights it with a visible blue ring/border
