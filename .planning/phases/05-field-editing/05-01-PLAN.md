# Plan 05-01: Inline Field Editing

## Overview

Implement inline editing for field values with full keyboard control.
Enter/click enters edit mode, Enter confirms, Escape cancels, Tab confirms+advances.
Edits increment `dataVersion`. Global shortcuts suppressed while editing.

## Steps

1. Create `src/components/EditableValue.tsx`
   - Span mode (not editing): shows value, click triggers onStartEdit
   - Input mode (editing): auto-focus, select-all, Enter/Escape/Tab handlers
   - stopPropagation on keydown to isolate from global handler

2. Update `src/components/FieldRow.tsx`
   - Accept `isEditing` and `dispatch` props
   - Replace raw value span with EditableValue
   - Wire callbacks: onStartEdit, onConfirm, onCancel, onTabConfirm
   - Enter on active row (when not editing) dispatches START_EDIT

3. Update `src/hooks/useKeyboardNavigation.ts`
   - Add Enter key: dispatch START_EDIT when active and not editing
   - Add +/- keys for zoom (gated behind !isEditing)
   - Ensure Space is also suppressed during editing (Phase 6 prep)

4. Verify reducer already handles CONFIRM_EDIT with dataVersion increment

5. Type-check and build

## Verification

- `npx tsc --noEmit` passes
- `npm run build` succeeds
- Manual: Enter on active field shows input, Enter confirms, Escape reverts, Tab confirms+advances
