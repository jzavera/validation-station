# Plan 06-01: Wire Space Key to CONFIRM_FIELD

## Goal
Complete Phase 6 (Field Validation) by wiring the Space key in useKeyboardNavigation to dispatch CONFIRM_FIELD. All other validation infrastructure (reducer actions, Tab skip logic, Confirm All button) is already in place from earlier phases.

## Requirements Addressed
- VAL-01: Space on active (non-editing) field marks it as confirmed
- VAL-02: Confirming sets validationSource to "user" (handled by CONFIRM_FIELD in reducer)
- VAL-03: Tab skips confirmed fields (NEXT_FIELD skip logic already in reducer)
- VAL-04: Confirm All button works (already in Header.tsx)

## Change

**File**: `src/hooks/useKeyboardNavigation.ts`

**Before** (lines 53-60):
```typescript
case ' ': {
  // Reserved for Phase 6: CONFIRM_FIELD
  // Prevent default to avoid page scrolling
  if (state.activeFieldId !== null) {
    e.preventDefault();
  }
  break;
}
```

**After**:
```typescript
case ' ': {
  if (state.activeFieldId !== null) {
    e.preventDefault();
    dispatch({ type: 'CONFIRM_FIELD' });
  }
  break;
}
```
